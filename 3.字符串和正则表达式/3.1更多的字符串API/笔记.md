# Es6新增字符串API    
以下均为字符串实例方法
- includes()  
判断字符串是否包含指定的子字符串  
2个参数： 子字符串 , 要查找的位置(可选)  
返回一个布尔值。调用的字符串包含给定的字符串返回true,否则返回false。  
当查询子字符串不需要其位置,只判断是否包含时使用该函数,如果需要知道子字符串的位置使用`indexOf()`或`lastIndexOf()`  

- startWith()  
判断字符串是否以指定的字符串开头  
2个参数： 字符串 , 要查找的位置/要开始的位置(可选)  
返回一个布尔值。  
该方法相当于判断`indexOf()`是否等于0.  

- endWith()  
判断字符串是否以指定的字符串结尾    
2个参数： 字符串 , 要查找的位置/要开始的位置(可选)  
返回一个布尔值。  

- repeat()  
参数：Number(重复的次数)  
将字符串重复指定次数，返回新的字符串
